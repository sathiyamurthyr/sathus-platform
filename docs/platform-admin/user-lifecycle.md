# User Lifecycle Management (Story 15.4)

User onboarding, invitation dispatch, session security, and identity governance.

## Features
- **User Directory**: Centralized directory listing active, pending, suspended, and terminated enterprise users.
- **Invitation Wizard**: Email invitation dispatch modal with role assignment and tenant scoping.
- **Session Manager & Device Audit**: Live audit of active user sessions (IP, Browser, Location) with 1-click session revocation trigger.
