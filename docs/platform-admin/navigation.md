# Administration Navigation Directory

Navigation structure for Sathus Cloud Administration.

## Route Map
- `/app/admin`: Executive Administration Dashboard Overview.
- `/app/admin/tenants`: Tenant provisioning and tiering management.
- `/app/admin/workspaces`: Workspace isolation settings and quota management.
- `/app/admin/users`: User directory, SSO, and MFA policies.
- `/app/admin/roles`: RBAC matrix and role definitions.
- `/app/admin/billing`: Subscription revenue and invoices.
- `/app/admin/settings`: Centralized platform settings framework.
