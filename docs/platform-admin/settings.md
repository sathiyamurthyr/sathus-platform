# Centralized Settings Framework

System-wide settings schema for Sathus Cloud Administration.

## Categories
- **Security**: Enforce MFA, session inactivity timeouts, IP whitelists.
- **AI Gateway**: Default model routing (Claude 3.5 Sonnet, GPT-4o), token rate limits.
- **Storage**: Capacity quota warning thresholds, SRE alert triggers.
- **Feature Flags**: Global toggles for experimental analytics, copilot, and RAG search features.
- **General**: System branding, default localization, contact parameters.
