# Enterprise Security Center (Story 15.6)

Centralized security posture, threat mitigation, and authentication policy engine.

## Key Features
- **Security Dashboard**: Live security health score (96%), MFA adoption rate (92.4%), active sessions count, and 24h blocked login sentinel triggers.
- **Authentication & Session Policies**: Mandatory MFA toggle, session idle timeout slider (e.g. 30 mins), and account lockout threshold slider.
- **IP Allow & Block Rules**: Subnet whitelist (`103.15.24.0/24`) and threat IP blacklist manager.
- **Trusted Devices**: Registry of verified user hardware (macOS, Windows, iOS) with 1-click device revocation.
