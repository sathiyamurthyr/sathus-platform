# White Label & Custom Branding Platform (Story 15.12)

Tenant-level white-labeling, brand asset management, and custom domains.

## Features
- **Brand Assets**: Custom logo uploads, dark mode variants, favicons, and splash messages.
- **Color Palette**: HSL brand color picker for primary accent and secondary highlight colors.
- **Custom Domains**: Domain alias mapping (`cloud.acmeglobal.com`) with automated SSL TLS certificate management.
- **Theme Preview**: Live interactive login screen and email template preview.
