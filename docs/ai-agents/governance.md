# Agent Governance Center (Story 27.13)

Enterprise AI compliance matrix tracking tenant trust scores, policy versioning, and compliance audit logs.

## Features
- **Trust Score Index**: Calculates tenant trust score based on policy compliance and zero-leakage records.
- **Policy Versioning**: Maintains version history and rollback capabilities for security rules.

## APIs
- `GET /api/v1/agent-governance/trust-score` — Fetch trust score and risk assessment metrics.
