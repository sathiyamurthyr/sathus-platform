# Multi-Agent Workflows & Safety Guardrails (Story 27.4)

Visual DAG orchestration engine, safety guardrails, prompt injection filters, and RBAC isolation policies.

## Features
- **Multi-Agent DAG Workflows**: Visual pipeline canvas chaining security triage, SRE remediation, and BI reporting agents.
- **Context & Memory Propagation**: Downstream node key-value output mapping (`outputKey`) passed into next step's working context window.
- **AI Safety & Defense System**: Real-time prompt injection detection, PII redaction, zero-trust tenant data scoping, and cost-cap per task throttles.
- **Audit Log Verification**: Compliance log records for all intercepted prompt injection attempts and PII masking events.
