# AI Agent Registry & Lifecycle Management (Story 27.2)

Centralized discovery catalog, registration wizard, and version manager for AI agents.

## Features
- **Agent Registry Catalog**: Category filtering (`analytics_bi`, `devops_sre`, `security_audit`, `customer_support`).
- **1-Click Agent Provisioning**: Registration modal specifying agent name, model kernel, temperature, and attached capabilities.
- **Versioning Control**: Version tracking (`v1.0.0`, `v2.1.0`) with active version switches.
- **Agent Status Lifecycle**: `ACTIVE` vs `SUSPENDED` status toggle.
