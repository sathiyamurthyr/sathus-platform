# Agent Performance Analytics (Story 27.14)

Performance analytics engine tracking reasoning quality scores, task success rates, and business impact metrics.

## Features
- **Agent Leaderboard**: Ranks agents by task success rate, reasoning score, and user satisfaction.
- **Cost Analytics**: Tracks 24h token spending per agent.

## APIs
- `GET /api/v1/agent-analytics/leaderboard` — Retrieve performance analytics leaderboard.
