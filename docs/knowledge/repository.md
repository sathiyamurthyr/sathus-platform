## Overview

The **Enterprise Knowledge Repository** manages knowledge collections, spaces, document libraries, snapshots, version histories, activity timelines, and asynchronous import/export pipelines.

---

## Key Features

- **Collections & Spaces**: Group domain knowledge into hierarchical tree structures.
- **Snapshots**: Point-in-time point recovery and state freezing.
- **Timeline**: Audit trail of every document mutation, indexing run, and graph update.
- **Import & Export**: Asynchronous batch job runners supporting bulk archive packaging.
